# SA Cowork MCP Remote OAuth

Use the bundled `cowork-remote-oauth-kernel` remote MCP connector as a real OAuth smoke test.

Guide the user to:

- open Connectors
- locate `cowork-remote-oauth-kernel`
- click `Connect`
- complete the browser OAuth flow if Cowork offers it

After that, summarize whether the connector shows as connected and whether its tools become available.
